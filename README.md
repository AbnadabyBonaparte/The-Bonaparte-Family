# The Bonaparte Family Portal

Portal oficial da família Bonaparte — história, genealogia, legado, fé, saúde, educação, expedição "Volta ao Mundo" e muito mais.

## Seções principais
- Home / StartHere
- Family & Legacy
- Expedition (mapa interativo)
- Faith, Health, Education, Work, Store
- ALSHAM (integração especial)

## Tecnologias
- Vite + React + TypeScript (client)
- Backend Node (server)
- Componentes UI customizados e Shadcn-like
- Documentação completa em `docs/` (inclui Dossiê de Arquitetura V1.1)
